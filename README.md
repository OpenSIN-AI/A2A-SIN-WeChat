# A2A-SIN-WeChat

> A2A agent for WeChat integration within the OpenSIN ecosystem.

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](LICENSE)
[![Stars](https://img.shields.io/github/stars/OpenSIN-AI/A2A-SIN-WeChat)](https://github.com/OpenSIN-AI/A2A-SIN-WeChat)

## Overview

This repository provides an Agent-to-Agent (A2A) integration for WeChat, enabling seamless communication and automation within the OpenSIN-AI ecosystem.

## Quick Start

```bash
git clone https://github.com/OpenSIN-AI/A2A-SIN-WeChat.git
cd A2A-SIN-WeChat
npm install
npm start
```

## Features

- A2A protocol support for WeChat
- Automated message routing
- Integration with OpenSIN MCP servers
- Full observability via OpenSIN-Ledger

## Architecture

This agent connects to the OpenSIN orchestrator and handles WeChat-specific messaging through the A2A protocol.
