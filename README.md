# A2A-SIN-WeChat

> A2A agent integration for WeChat messaging platform.

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](LICENSE)
[![Stars](https://img.shields.io/github/stars/OpenSIN-AI/A2A-SIN-WeChat)](https://github.com/OpenSIN-AI/A2A-SIN-WeChat)

## Overview

A2A (Agent-to-Agent) integration for WeChat. Part of the OpenSIN-AI ecosystem.

## Quick Start

```bash
git clone https://github.com/OpenSIN-AI/A2A-SIN-WeChat.git
cd A2A-SIN-WeChat
npm install
npm start
```

## Features

- A2A protocol support
- WeChat messaging integration
- Automated agent communication
- OpenSIN MCP server compatibility

## Architecture

This agent connects to the OpenSIN orchestrator via A2A protocol and handles WeChat messaging operations.
